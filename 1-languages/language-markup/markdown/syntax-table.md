# Syntax - `Table`

## Align Columns

## Left-Aligned Columns

**Type 1: Default**
Example: Syntax

```markdown
| Column Title 1 | Column Title 2 |
| -------------- | -------------- |
| Content        | Content        |
```

Example: Output

| Column Title 1 | Column Title 2 |
| -------------- | -------------- |
| Content        | Content        |

**Type 2: Colon**
Example: Syntax

```markdown
| Column Title 1 | Column Title 2 |
| :------------- | :------------- |
| Content 1      | Content 2      |
```

Example: Output

| Column Title 1 | Column Title 2 |
| :------------- | :------------- |
| Content 1      | Content 2      |

## Right-Aligned Columns

Example: Syntax

```markdown
| Column Title 1 | Column Title 2 |
| -------------: | -------------: |
| Content 1      | Content 2      |
```

Example: Output

| Column Title 1 | Column Title 2 |
| -------------: | -------------: |
| Content 1      | Content 2      |

## Center-Aligned Columns

Example: Syntax

```markdown
| Column Title 1 | Column Title 2 |
| :------------: | :------------: |
| Content 1      | Content 2      |
```

Example: Output

| Column Title 1 | Column Title 2 |
| :------------: | :------------: |
| Content 1      | Content 2      |
