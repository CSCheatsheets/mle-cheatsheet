# Syntax - `Table`

## Align Columns

## Default

By default, column titles are center-aligned, and column contents are left-aligned.

<u>Example</u>: Syntax

```markdown
| Column Title 1 | Column Title 2 |
| -------------- | -------------- |
| Content        | Content        |
```

<u>Example</u>: Output

| Column Title 1 | Column Title 2 |
| -------------- | -------------- |
| Content        | Content        |

## Left-Aligned Columns

<u>Example</u>: Syntax

```markdown
| Column Title 1 | Column Title 2 |
| :------------- | :------------- |
| Content 1      | Content 2      |
```

<u>Example</u>: Output

| Column Title 1 | Column Title 2 |
| :------------- | :------------- |
| Content 1      | Content 2      |

## Right-Aligned Columns

<u>Example</u>: Syntax

```markdown
| Column Title 1 | Column Title 2 |
| -------------: | -------------: |
| Content 1      | Content 2      |
```

<u>Example</u>: Output

| Column Title 1 | Column Title 2 |
| -------------: | -------------: |
| Content 1      | Content 2      |

## Center-Aligned Columns

<u>Example</u>: Syntax

```markdown
| Column Title 1 | Column Title 2 |
| :------------: | :------------: |
| Content 1      | Content 2      |
```

<u>Example</u>: Output

| Column Title 1 | Column Title 2 |
| :------------: | :------------: |
| Content 1      | Content 2      |
