# Command - `redirection`

## Redirect Console Outputs to Overwrite a File

```shell
# Command
[some_command] > [file_name]

# Example
echo "Hello World!" > ./hello.txt
```

## Redirect Console Outputs to Append to a File

```shell
# Command
[some_command] >> [file_name]

# Example
echo "Hello World!" >> ./hello.txt
```
